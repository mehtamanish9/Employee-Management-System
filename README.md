# Employee Management System

A Java Swing desktop application for managing employee records using the MVC architecture.

## Features
- Add, update, and delete employee records
- Auto-save and load data between sessions (Java Serialization)
- Interactive GUI with table-based record display

## Tech Stack
Java · Java Swing · AWT · Collections Framework · Serializable · MVC Pattern

## Project Structure
EMS/src/
├── model/Employee.java
├── controller/Manager.java
└── view/TableModel.java, SystemGUI.java

## How to Run
1. Clone the repo
2. Open in any Java IDE (IntelliJ / Eclipse)
3. Run `SystemGUI.java`
